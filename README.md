# shooter-b-privacy

Privacy policy website for DuckOff, hosted via GitHub Pages.

## Live URL

https://atelabro.github.io/alab-game-common/

## Edit placeholders

Open `index.html` and replace:
- `[APP NAME]`
- `[DEVELOPER NAME]`
- `[EFFECTIVE DATE]` (e.g. March 11, 2026)
- `[CONTACT EMAIL]`
- `[YOUR GITHUB USERNAME]` in this README

## Publish

1. Push this repo to GitHub as `shooter-b-privacy`.
2. Go to repo Settings > Pages.
3. Set Source to "Deploy from a branch", branch `main`, folder `/ (root)`.
4. Save. The live URL will appear within a few seconds.

## App Store Connect

Use this URL in the Privacy Policy field:

https://atelabro.github.io/alab-game-common/
